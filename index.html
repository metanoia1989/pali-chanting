技术栈

纯静态架构，零服务端依赖，极致轻量与隐私安全。

桌面端 → Electron + Vite
前端 → Vue3
音频引擎 → Web Audio API
数据格式 → { word, start_time, end_time } JSON
移动端 → PWA (Service Worker)
托管 → Cloudflare Pages
