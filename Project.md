📿 Pāli Chant: 巴利语唱诵精听与发音学习工具

产品概述
Pāli Chant 是一款专为巴利语（Pāli）学习者、南传佛教修行者及语言研究者设计的轻量级唱诵辅助工具。针对巴利语“口耳相传、重音律节奏”的学习特点，本应用将传统音频播放与文本深度绑定，提供单词级精准同步、发音重复精听、沉浸式跟读等核心能力。

产品采用 Electron (桌面端) + PWA (移动端) 双端架构，纯静态部署，零服务端依赖，确保离线可用、隐私安全与极致的轻量化体验。

核心技术栈
层级   技术选型   选型理由
桌面框架   Electron + Vite   跨平台桌面支持，Vite 构建极速，开发体验流畅

前端核心   Vanilla JS / Vue3 (可选)   无重型框架负担，DOM 操作直接，保障高亮渲染性能

音频引擎   Web Audio API + HTML5 Audio   原生音频控制，支持精确到毫秒级的 currentTime 跳转

数据格式   自定义 JSON 时间戳   { word, start, end } 结构，实现单词级音文同步

移动端方案   PWA (Service Worker)   纯静态托管，免上架审核，支持离线缓存与主屏幕安装

静态托管   Cloudflare Pages   全球 CDN 加速，国内直连友好，自动 HTTPS，永久免费

状态管理   响应式状态机   统一管理播放/暂停/循环/手动跳转四种交互状态

核心功能点

🎯 音文同步与可视化
单词级精准高亮：基于预标注的 JSON 时间戳，音频播放时逐词高亮，彻底解决巴利语长句连诵时“找不到读到哪”的痛点。
三态视觉反馈系统：
    🔵 正在播放：高亮当前单词，引导视觉焦点。
    ⚫ 未播放：默认黑色，保持文本可读性。
    ⚪ 已播放：自动变灰，直观呈现学习进度；手动跳转时智能重置灰色状态，避免认知误导。
单词级进度条：底部进度条支持拖拽吸附至最近单词起点，悬停预览对应文本，告别传统进度条“拖不准”的问题。

🎧 四维交互精听体系
针对不同学习场景，提供无缝切换的播放控制：
操作   行为   适用场景
单击单词   跳转至该词播放，播完自动继续   快速确认某个词的发音、重音位置

双击单词   进入该词循环播放模式   反复模仿难发辅音、元音组合、唱诵旋律

R 键 / 🔁按钮   重播当前高亮单词   键盘党友好，不打断手部记录笔记的动作

空格键   全局播放 / 暂停   最基础的播放控制，已阻止页面滚动默认行为

ESC 键   退出循环模式   明确的状态切换，防止误入循环无法退出

📜 巴利语专属优化
唱诵段落结构化：支持按偈颂（Gāthā）分段展示，保留巴利语文本的韵律换行，而非简单堆砌为纯文本块。
转写字符完整支持：内置 Noto Sans Pali / Adwaita Mono 等字体回退链，确保 ṃ, ṇ, ṭ, ḍ, ū, ī 等特殊转写字母在所有设备上正确渲染，无乱码、无缺字。
低延迟音频响应：预加载所有唱诵音频至内存，点击单词到发声的延迟控制在 50ms 以内，保障跟读节奏不被打断。

产品特性与优势

🔌 纯离线可用：所有音频、文本、逻辑均打包在本地。首次加载后，断网状态下仍可完整使用全部功能，适合禅修中心、寺院等无网络环境。
🪶 极致轻量：桌面端安装包 < 80MB，移动端 PWA 首屏加载 < 1s。无广告、无追踪、无账号体系，打开即用。
💰 零运维成本：纯静态架构 + Cloudflare Pages 托管，无服务器、无数据库、无 SSL 证书费用。开发者可专注于内容迭代而非基础设施维护。
🔒 隐私与安全：无任何网络请求，不收集用户行为数据，不上传任何本地文件。学习记录仅保存在浏览器 localStorage 中。
📱 跨端一致体验：Electron 桌面端与 PWA 移动端共享同一套前端代码，交互逻辑、视觉样式、音频同步精度完全一致，无需为不同设备重新学习操作。
