// 佛随念
1. 如是世尊，为阿罗汉 - 应供者，
2. 正等正觉者，
3. 明行具足者，
4. 善逝者，
5. 世间解者，
6. 无上调御丈夫，
7. 天人师，
8. 佛陀，
9. 世尊。

// 法随念
1. 世尊善说 - 此达摩，
2. 现见，
3. 即时，
4. 请来亲见，
5. 导向解脱，
6. 智者各自证知。

// 僧随念
1. 善行具足 - 世尊之弟子众，
2. 直行具足 - 世尊之弟子众，
3. 正行具足 - 世尊之弟子众，
4. 和敬行具足 - 世尊之弟子众。此四双 - 八辈 -
即世尊之弟子众。
5. 应受供养者，
6. 应受款待者，
7. 应受布施者，
8. 应受礼敬者，
9. 为世间无上福田。

// 礼敬七佛
1. 礼敬毗婆尸佛 - 具眼智者 光耀庄严
礼敬尸弃佛 - 慈念一切众生
2. 礼敬毗舍浮佛 - 已除诸漏 有大精进
礼敬拘留孙佛 - 破除魔军者
3. 礼敬拘那含牟尼佛 - 已除诸漏
梵行清净 礼敬迦叶佛 - 一切解脱自在
4. 礼敬鸯伽罗 - 乔达摩佛 - 释迦族之子
具足光明 - 宣说正法 - 除灭一切苦
5. 世间诸有 - 已灭烦恼焰 - 如实观照 - 彻见真相
彼等从不诽谤 - 乃具足无畏之伟人
6. 乔达摩佛 - 天人亲爱
明行具足 - 伟大无畏
我等礼敬 - 无上佛陀

// 礼敬佛塔与菩提树
我以至诚礼敬 - 一切佛塔 - 于一切处 - 所安立者
舍利 - 大菩提树 - 及诸佛像 恒时礼敬
我礼敬 - 此菩提树
于其下坐 - 导师证得正觉 - 降伏一切烦恼
我亦礼敬 - 此大菩提树
曾为 - 世间依怙所赞叹 - 我顶礼汝 - 菩提树王

善哉！善哉！善哉！
